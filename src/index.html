Latest projects

Pulse Tracker SDK

The Pulse tracker is one of the key components in Schibsteds data strategy. It is used to collect all events from Schibsted web components and sites. It allowes the different sites to describe and collect data about user behaviour in a compatible way accross the different site properties.

Written in vanilla javascript as a collection of multiple NPM modules.

Pulse Monitor

Pulse Monitor gives site developers and tracking implementers and insight in exactly what data is being sent to Schibsted. A key part of the data strategy of schibsted is to collect compatible data cross site. To do this a schema was developed and Pulse monitor will tell you to what degree the data you are sending is compatible with the schema.

Written in React.js, Redux and redux-saga.

Pulse Unicorn

Pulse Unicorn is a Chrome Developer Tools Extension that gives you similar information that Pulse monitor will give you, but in realtime inside chrome dev tools.

Written in React.js and Redux.

Schibsted Messenger

Schibsted Messenger is a core component used across all Classified sites of Schibsted. It is a fully featured instant messaging sollution that can be dropped in to web pages or iOS and Android apps.

Written in Vue.js and vuex.
